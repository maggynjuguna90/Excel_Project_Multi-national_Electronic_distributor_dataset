# Excel_Project_Multi-national_Electronic_distributor_dataset
This project involved cleaning, analysing and creating insights using a dynamic Dashboard for a multi-regional electronic distributor.
The data contains sales record with the following main columns :Order ID, City,Country,region,salesperson,quantity,Unit cost, revenue,Gross profit.

**ISSUES FOUND**
The data contained one duplicate.
Some values were negative.
Some required dates were earlier than the order dates.
Some values were missing from the record.

**ASSUMPTIONS MADE.**
The minimum lead time days is 5 days.

**DATA CLEANING** 
1 duplicate was removed from the data.
Correct formatting of columns was done.
Trimming was done to remove spaces.
Missing values were replaced with 0.
Negative values were turned into positive values using find and replace.

**TOOLS USED**
Pivot tables and charts
Excel formulas and Functions
Slicers
